# WebDev101

A playground & learning lab for modern web development — diving deep into Svelte, SvelteKit, TypeScript, GSAP animations, shaders, PocketBase, and all the juicy frontend/back stuff in between.

---

## What’s inside?

- **Svelte & SvelteKit**  
Building reactive, fast UIs with the slickness of Svelte and the power of SvelteKit routing + SSR.

- **TypeScript**  
Typed JavaScript goodness for safer, scalable code.

- **GSAP**  
Smooth, performant animations that bring your UI to life.

- **Shaders**  
Playing with GLSL and WebGL for cool visual effects and graphics magic.

- **PocketBase**  
Lightweight backend for auth, realtime DB, and file storage — zero hassle.

- **And yada yada**  
Other cool webdev tools, hacks, and experiments as I go.
